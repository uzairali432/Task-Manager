# Task Manager App

A full-stack Task Manager application built with **MERN (MongoDB, Express, React, Node.js)** stack. This app allows users to **register, login, add, update, delete, and manage tasks** efficiently. It includes **JWT authentication** and **RESTful APIs** for secure and seamless task management.

---

## Features

- **User Authentication**
  - Register and login with email and password.
  - Passwords are securely hashed.
  - JWT-based authentication for protected routes.

- **Task Management**
  - Add new tasks with title and description.
  - Edit and update existing tasks.
  - Delete tasks.
  - Mark tasks as completed.
  - View all tasks for logged-in user.

- **Backend**
  - Built with **Node.js** and **Express.js**.
  - RESTful API routes for user and task management.
  - MongoDB database to store user and task data.

- **Frontend**
  - Responsive UI built with **React.js**.
  - Interactive task dashboard.
  - Forms for login, registration, and task creation.

- **Security**
  - JWT tokens for authentication.
  - Password hashing with bcrypt.
  - Protected routes to ensure only authenticated users can access tasks.

---

## Technologies Used

- **Frontend:** React.js, HTML5, CSS3, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT (JSON Web Tokens)  
- **Password Security:** bcrypt
